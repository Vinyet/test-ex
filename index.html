Ejercicios

  1. Devuelve el nombre de cada película utilizando forEach.
    SOLUCIÓN 
                        
    movies.forEach(film => film.title)
  2. Devuelve el nombre de las películas que lanzadas antes de 2005.
    SOLUCIÓN 
                        
    movies.forEach((film) => { 
        film.releaseDate < 2005 ? film.title : null;
    });
